Project Overview

BookSky is a web-based book management application developed to manage and organize book information efficiently. The application allows users to add, view, and delete book records through an interactive user interface.

Key Features

Create and manage book records dynamically

Display book details in real time using JavaScript

Delete book entries with user interaction

Responsive and user-friendly web interface

Technologies Used

HTML5

CSS3

JavaScript (DOM Manipulation, Event Handling)

Functional Workflow

User inputs book title and description

Application processes data using JavaScript

Book details are rendered dynamically on the UI

Users can remove book entries as required

Skills Demonstrated

Frontend web development

DOM manipulation using JavaScript

Event-driven programming

UI design and layout implementation

Use Case

This project demonstrates the implementation of a CRUD-based frontend application suitable for beginners and entry-level software development roles.

Future Improvements

Add edit and update functionality
