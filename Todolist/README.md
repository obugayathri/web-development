Project Description

To-Do List is a simple and user-friendly web application that allows users to add, view, and manage daily tasks efficiently. This project helps users organize their tasks in a structured way using a clean and responsive interface.

Features

Add new tasks dynamically

View all tasks in a list format

Mark tasks as completed

Delete tasks when no longer required

Technologies Used

HTML

CSS

JavaScript

How It Works

User enters a task in the input field

Task is added dynamically to the task list

User can mark tasks as completed

Tasks can be deleted with a single click

Project Use Case

This project helps beginners understand:

DOM manipulation using JavaScript

Event handling

Building interactive frontend applications

Future Enhancements

Edit task functionality
